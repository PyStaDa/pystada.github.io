	* Messy datasets? Missing values? missingno provides a flexible and easy-to-use missing data matrix (nullity matrix?) visualization that allows you to get a quick visual summary of the completeness (or lack thereof) of your dataset. It's built using matplotlib, so it's fast, and takes any DataFrame input that you throw at it, so it's flexible. Just pip install missingno to get started.
		https://github.com/ResidentMario/missingno
